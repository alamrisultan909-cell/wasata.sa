# مؤسسة المنصة الشاملة للوساطة - wasata.sa

موقع تعريفي احترافي لمؤسسة المنصة الشاملة للوساطة - متخصصة في الخدمات الاستثمارية والعقارية والاستشارية.

## 🌐 الميزات

- ✨ تصميم حديث واحترافي مع تأثيرات بصرية جذابة
- 📱 استجابة كاملة لجميع أحجام الشاشات (Responsive Design)
- 🎨 واجهة مستخدم جميلة بألوان متناسقة
- ⚡ أداء سريع وتحميل فوري
- 🔍 محسّن لمحركات البحث (SEO Friendly)
- 🌍 دعم كامل للغة العربية (RTL)

## 📋 المحتوى

### الأقسام الرئيسية

1. **الرئيسية (Hero Section)**
   - عنوان جذاب مع دعوة للعمل
   - خلفيات متدرجة ومتحركة

2. **الخدمات**
   - أنشطة الاستثمار للحساب الخاص
   - الوساطة العقارية
   - إعادة تأهيل وهيكلة العمليات
   - الاستشارات الإدارية العليا
   - خدمات تسويقية
   - أنشطة الخدمات العامة

3. **عن الشركة**
   - الرؤية والرسالة
   - التراخيص والشهادات الرسمية

4. **اتصل بنا**
   - نموذج تواصل تفاعلي
   - معلومات الاتصال

## 🚀 التثبيت والاستخدام

1. استنسخ المستودع
```bash
git clone https://github.com/alamrisultan909-cell/wasata.sa.git
```

2. افتح الملف في متصفحك
```bash
open index.html
```

أو استخدم خادم محلي:
```bash
python -m http.server 8000
```

ثم توجه إلى `http://localhost:8000`
